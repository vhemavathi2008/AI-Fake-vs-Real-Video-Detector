# AI Fake vs Real Video Detector (Prototype)

## About
I developed this project as part of my learning journey in web development and AI concepts.  

With the increasing use of AI-generated content and deepfake videos on social media, I became curious about how we can differentiate between real and AI-generated videos. This led me to build a simple prototype to explore this concept.

## Technologies Used
- HTML
- CSS
- JavaScript

## Key Features
- Video upload and preview
- Basic detection logic
- Confidence score with reasoning

## How to Run
1. Open `index.html` in your browser.
2. Upload a video or image to test detection.

## Notes
This is a prototype system and not a fully trained AI model. I plan to improve it further using machine learning techniques.  
I also used AI tools to assist me in learning and developing this project.

## Future Scope
- Add simple social media indicators (red for AI-generated, green for real) to help users identify content.  
- Improve detection accuracy using AI/ML techniques.
